# VicharManthan Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

Application runs on `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- React Router for navigation
- Context API for state management
- Socket.io for real-time features
- Axios for API calls
- Responsive design
- Modern UI with Lucide icons

## Pages

- **Home** - Landing page
- **Login/Register** - Authentication
- **Ideas** - Browse all ideas
- **Idea Detail** - View idea details, comments, team
- **Create Idea** - Post new idea
- **Profile** - User profile management
- **Proposals** - Track your applications
- **Chats** - Real-time messaging
- **Notifications** - Activity updates

## Components

- **Navbar** - Navigation with notifications badge
- **IdeaCard** - Idea preview card with upvote
- **AuthContext** - Authentication state
- **SocketContext** - Real-time connection
